Requires Android 10 or higher

How to Install

  1. On your Android device, go to Settings → Security and enable "Unknown sources" (or "Install unknown apps")
  2. Download the APK from the link above
  3. Open the downloaded APK file to install
  4. Launch "Magic App" from your app drawer
